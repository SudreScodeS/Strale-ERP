🔐 Autenticação

Todas as rotas protegidas requerem header Authorization: Bearer <token>

POST /api/v1/auth/login

Realiza login e retorna JWT.

Request:

curl -X POST /api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "senha123"}'

Response (200):

{
  "success": true,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "user": { "id": "uuid", "username": "admin", "role": "admin" }
  }
}

POST /api/v1/auth/register 🔒 Admin

Cria novo usuário.

Request:

curl -X POST /api/v1/auth/register \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <admin-token>" \
  -d '{"username": "vendedor1", "email": "v1@email.com", "password": "senha123", "role": "seller"}'

POST /api/v1/auth/refresh

Renova token de acesso.

POST /api/v1/assistant 🔒 Auth

Envia mensagem para o assistente IA (Ollama).

Request:

  -H "Content-Type: application/json" \
