# AGENTS.md

## Contexto do projeto

Este projeto é um ERP/SaaS para TCC, com foco em gestão empresarial, produtos, estoque, pedidos, clientes, financeiro e IA integrada.

O objetivo é parecer uma startup real, moderna e viável, não um template genérico feito por IA.

## Regras gerais

- Antes de alterar arquivos, analise a estrutura.
- Não refatore o projeto inteiro sem necessidade.
- Não adicione dependências sem justificar.
- Não crie páginas novas se puder resolver com componentes existentes.
- Não remova componentes sem explicar.
- Preserve a identidade visual atual.
- Priorize responsividade em 1920x1080.
- Melhore UI/UX sem deixar aparência genérica.
- Sempre explique quais arquivos serão alterados e por quê.
- Após alterar, indique como testar.

## Sobre a IA de Produtos

A IA de Produtos deve parecer uma funcionalidade real dentro do ERP.

Ela pode trabalhar inicialmente com dados mockados, simulando:
- produtos com estoque crítico;
- produtos parados;
- sugestão de reposição;
- análise de margem;
- produtos mais vendidos;
- produtos com baixa saída;
- recomendações de preço;
- insights operacionais.

Não integrar API externa de IA sem autorização.

## Estilo visual

- Visual premium, moderno e limpo.
- Evitar cards genéricos demais.
- Evitar gradientes.
- Evitar seções vazias.
- Usar hierarquia visual clara.
- Manter o produto com aparência SaaS profissional.
